## 💳 Alke-Wallet

###📌 Creación de una página web básica para comprender los fundamentos del desarrollo web.  

---

## 🚀 Funcionalidades

- Registro e inicio de sesión de usuarios
- Protección de rutas (acceso solo con sesión activa)
- Depósitos de dinero
- Envío de dinero entre usuarios
- Visualización de saldo disponible
- Historial de movimientos
- Interfaz moderna con efecto *glassmorphism*
